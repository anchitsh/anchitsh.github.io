Fears to Fathom - Home Alone

Year: 2024
Role: Lead Developer, Mobile Port
Tech: Unity, C#, HLSL, Blender, Photoshop
Platforms: iOS, Android

What I did

I led the mobile port of Fears to Fathom - Home Alone, adapting the original PC game for touch controls and lower-powered mobile devices.

Highlights

  • Implemented touch gestures and on-screen controls for mobile play.
  • Reduced asset, texture, script, and draw-call cost for mobile performance.
  • Used LODs, mesh combining, texture atlases, object pooling, and Addressables to improve runtime behavior.

The mobile release was available for several months on Android and iOS before being discontinued.
